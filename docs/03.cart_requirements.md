# 장바구니 요구사항 분석 및 기능 목록
## 기본 정보
### 프로젝트명:
패션 쇼핑몰 장바구니 페이

### 사용하게 될 기술:
React

### 문서 작성자: 100ke

## 📝 고객 요구사항 정리
- 담긴 상품의 목록 표시
    - 각 상품의 상품명, 가격, 수량을 표시
- 수량에 따른 총 금액, 배송비 표시 (3000원)
- 10만원 이상 구매 시 배송비 무료
- 뒤로가기 버튼을 이용해 메인 상품 목록 페이지 이동
- 결제하기 버튼을 누르면 결제 모듈을 모달로 사용할 수 있도록 연동
- 중간 결과물을 확인할 수 있는 test URL 요청

## 📋 기능 목록
- 뒤로가기 버튼
- 결제하기 버튼
- 장바구니 페이지 `<Cart />`
    - 각 상품의 상품명, 가격, 수량 표시
    - 수량에 따른 총 금액, 배송비 표시
